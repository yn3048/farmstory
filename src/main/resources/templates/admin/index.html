관리자 메인

회원현황 +새창으로 더보기

회원아이디 이름 닉네임 휴대폰 이메일 등급 회원가입일
[[${user.uid}]] [[${user.name}]] [[${user.nick}]] [[${user.hp}]] [[${user.email}]] [[${user.role}]]

상품현황 + 새창으로 더보기

상품번호 상품명 구분 가격 재고 등록일
[[${product.pname}]] [[${product.cate}]] [[${product.price}]] [[${product.stock}]]

주문현황 + 더보기

주문번호 상품명 판매가격 수량 합계 주문자 주문일
